# Generating Functions

for a sequence of numbers:

$$a_1, a_2, a_3, ... a_n, ...$$

the corresponding generating function is:

$$f(x) = a_1x + a_2 x^2 + .. + a_n x^n + ...$$

$$ = \sum^{\infty}_{r = 0} a_r x^r$$

## Functions

| Area          | Function                                                                                   |

| Logical       | AND, IF, NOT, OR                                                                           |
| Lookup        | HLOOKUP, VLOOKUP                                                                           |

### LARGE

#### Syntax

```
LARGE(range, k)
```

- Specifies $k$ largest values
